--------Dependencies--------
actix-web (ADD: rustls)
actix-files
diesel (ADD: mysql sqlite postgres)
libsqlite3-sys (ADD: bundled)
argon2

----------Database----------
Artists:
-Id: u32 (for DB)
-Path: Path or String
-Name: String
-Website: Uri or String

Albums:
-Id: u32 (for DB)
-Name: String
-Year: Date or String
-Genre: String or Enum
-Path: Path or String

Songs:
-Id: u32 (for DB)
-Name: String
-Year: Date
-Explicit: Boolean
-Path: Path or String
-Album: u32 (for DB)

SongsArtists:
-Song: u32 (for DB)
-Artist: u32 (for DB)

Maintainers:
-Name: String
-Contact: Uri or String

Accounts:
-Username: String
-PhcHash: String

---------Structures---------
-Maintainer (inherit from Maintainers)
-Account (inherit from Accounts)
-Artist (inherit from Artists)
-Song (inherit from Songs + SongsArtists)
-Album (inherit from Albums)
-Hostinfo:
--Name: String
--Uuid: Uuid
--Website: Uri or String
--Maintainers: Array<Maintainer>
--Secured: Boolean
--Endpoints: Endpoints
-Endpoints:
--Metadata: Uri or String
--Metadata: Uri or String
--Audiodata: Uri or String

-----------Routes-----------
Hostinfo:
-Description: Information of Music Server
-Example: "/", "/hostinfo"
-Methods: GET
-Response: Hostinfo struct

Audiodata:
-Description: Get Audio Data
-Input: URLPath
-Example: "/audiodata/{ArtistDirname}/{AlbumDirname}/{SongFilename}"
-Methods: GET, HEAD
-Response: Audio Data in HTTP-Body, Metadata in HTTP-Head

-Example: "/audiodata/{ArtistDirname}/{AlbumDirname}"
-Response: Song List of Album

-Example: "/audiodata/{ArtistDirname}"
-Response: Album List of Artist

-Example: "/audiodata"
-Response: Artist List of Host

Metadata:
-Description: Retrieve Song, Album, Artist Metadata
-Example: "/metadata/{ArtistDirname}/{AlbumDirname}/{SongFilename}" (Same structure as Audio Data)
-Input: URLPath
-Methods: GET
-Response: Metadata of Item
